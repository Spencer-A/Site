Summary of Image Formats

  • GIF:
    • Low file size
    • Limited to 256 colors
    • Able to be animated
  • PNG:
    • Two types: PNG-8 and PNG-24
    • PNG-8 is essentially the same as a GIF (without animation capabilities)
    • PNG-24 is much higher quality and supports transparent backgrounds
    • Major downsides to PNG: high file size, and not supported by all major web browsers
  • JPG:
    • High range of colors avaliable
    • Best for complex images
    • Able to drag a quality slider to determine quality (and therfore file size)
